Overview

The Piconode is a 3-channel compact wireless data logger designed for long-term, unattended operations. It includes one configurable analog channel, one thermistor channel, and one pulse counter channel. The device transmits high-accuracy measurements via LoRa, while onboard software and mobile apps allow field setup, sample collection, and signal coverage testing without manual data retrieval.

What's Included

Robust components and intelligent services that make analog monitoring reliable from the start

  • 3-channel data logger (1 configurable, 1 thermistor, 1 pulse counter)
  • Sensor power output: 5 V DC, up to 50 mA
